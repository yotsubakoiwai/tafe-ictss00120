## Model Checkpoints

Due to GitHub file size limitations, trained model checkpoints are hosted separately on Google Drive.

Download the checkpoints from:

**[Google Drive Link](https://drive.google.com/drive/folders/1NSlxr619m81B493d8LEj4Nnx4JV1pRyN?usp=sharing)**

Extract or copy the downloaded `.pth` files into:

```text
saved_models/
```

No retraining is required. The notebooks will automatically load the provided checkpoints and reproduce the reported evaluation results.
