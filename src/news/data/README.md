# AG News Classification Dataset Download Instructions

1. Download from https://www.kaggle.com/datasets/amananandrai/ag-news-classification-dataset
2. Download both test.csv & train.csv 
3. Additionally, download glove.6B.300d.txt from https://www.kaggle.com/datasets/thanakomsn/glove6b300dtxt
4. Place all data files in the /news/data directory
