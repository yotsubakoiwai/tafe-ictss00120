# tafe-ictss00120
#### This repo consists of all projects related to TAFE work on ICTSS00120 - Artificial Intelligence Skill Set

## 
### 💻 Running the Notebook on a Local Machine

### Prerequisites

* Python 3.12+ (or the version used for development)
* Jupyter Notebook or JupyterLab

### Setup

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY.git
cd YOUR_REPOSITORY
```

Launch Jupyter Notebook:

```bash
jupyter notebook
```

or

```bash
jupyter lab
```

Open `notebook.ipynb` and run all cells sequentially.

### Saved Models

Pre-trained model checkpoints are available in the `saved_models/` directory and can be loaded directly for evaluation without retraining.

### Results

Generated outputs, evaluation metrics, and plots can be found in the `results/` directory.
